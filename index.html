安装指南

按照以下步骤轻松安装固件

1 准备工作

  • 确保您的设备是 x86_64 架构的兼容设备
  • 下载适合的固件版本（OpenClash、PassWall 或 HelloWorld）
  • 准备一个至少 1GB 的 USB 闪存盘或其他存储介质

2 写入固件

  • 使用 Etcher 或 Rufus 等工具将下载的固件写入存储介质
  • 确保写入模式正确，选择 DD 模式（如果使用 Rufus）

3 安装步骤

  • 将写入固件的存储介质插入目标设备并启动
  • 进入 BIOS/UEFI 设置，设置从 USB 设备启动
  • 启动后按照提示完成安装，可选择安装到硬盘或直接从 USB 运行
  • 安装完成后，设备将自动重启。默认 IP 地址为 192.168.1.1
  • 默认登录密码 password
